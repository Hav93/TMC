# 📚 资源监控功能使用指南

## 🎯 功能概述

资源监控功能可以自动监控Telegram群组/频道中的资源链接（115分享、磁力链接、ed2k），并自动转存到115网盘。

### 核心特性

- ✅ **自动捕获** - 实时监控消息中的资源链接
- ✅ **智能过滤** - 支持关键词包含/排除
- ✅ **去重机制** - 自动识别重复资源
- ✅ **消息快照** - 完整保存原始消息上下文
- ✅ **批量操作** - 一键转存/忽略/删除多个资源
- ✅ **标签系统** - 灵活的资源分类管理

---

## 🚀 快速开始

### 1. 创建监控规则

1. 进入 **资源监控** 页面
2. 点击 **新建规则** 按钮
3. 填写规则信息：

```
规则名称: 电影资源
监控来源: -1001234567890 (群组ID，多个用逗号分隔)
包含关键词: 4K, 蓝光 (可选)
排除关键词: 枪版, CAM (可选)
链接类型: ☑ 115分享 ☑ 磁力 ☑ ed2k
目标路径: /电影
自动转存: ☑ 是
```

4. 点击 **确定** 保存规则

### 2. 查看捕获的资源

1. 切换到 **资源记录** 标签
2. 查看已捕获的资源列表
3. 点击 **详情** 查看完整信息
4. 对于待处理的资源，可以：
   - 点击 **转存** 立即转存到115
   - 批量选择后使用 **批量转存**
   - 使用 **批量忽略** 跳过不需要的资源

## 📖 详细说明

### 监控规则配置

#### 监控来源
- 输入群组/频道的ID（数字）
- 多个ID用逗号分隔
- 如何获取群组ID：
  1. 在Telegram中打开群组
  2. 查看群组链接：`https://t.me/c/1234567890/...`
  3. ID为：`-1001234567890`（加上 -100 前缀）

#### 关键词过滤
- **包含关键词**：消息必须包含其中任一关键词才会被捕获
- **排除关键词**：包含这些关键词的消息会被忽略
- 多个关键词用逗号分隔
- 留空表示不过滤

#### 链接类型
- **115分享链接**：`https://115.com/s/xxx`
- **磁力链接**：`magnet:?xt=urn:btih:xxx`
- **ed2k链接**：`ed2k://|file|xxx`

#### 目标路径
- 指定115网盘的保存路径
- 资源会保持原有文件夹结构转存到此路径下
- 示例：
  - `/电影` - 所有电影资源
  - `/剧集/日剧` - 日剧资源
  - `/` - 根目录

#### 自动转存
- **启用**：捕获后立即自动转存到115
- **禁用**：需要手动确认后才转存

---

### 资源记录管理

#### 状态说明
- **待处理**：已捕获，等待转存
- **已转存**：成功转存到115
- **失败**：转存失败（查看错误信息）
- **已忽略**：手动忽略的资源

#### 批量操作
1. 选择多条记录（勾选复选框）
2. 点击批量操作按钮：
   - **批量转存**：将选中的资源转存到115
   - **批量忽略**：标记为已忽略
   - **批量删除**：从记录中删除

#### 搜索和筛选
- **搜索**：在消息内容中搜索关键词
- **状态筛选**：按状态过滤记录
- **规则筛选**：按监控规则过滤

## ⚙️ 高级功能

### 消息快照
每条资源记录都保存了完整的消息信息：
- 原始消息文本
- 发送者信息
- 群组信息
- 发送时间
- 浏览/转发数
- 媒体文件信息

### 去重机制
系统会自动识别重复的资源：
- 基于链接内容生成唯一指纹
- 相同链接只会被捕获一次
- 避免重复转存浪费空间

### 标签系统（即将上线）
- 为资源添加自定义标签
- 按标签筛选和管理
- 跨规则的资源分类

## 🔧 故障排除

### 问题1：资源没有被捕获
**可能原因：**
- 规则未启用（检查规则状态）
- 关键词过滤太严格
- 群组ID不正确
- 消息中没有链接

**解决方法：**
1. 确认规则状态为"启用"
2. 检查关键词设置
3. 验证群组ID是否正确
4. 查看系统日志

### 问题2：转存失败
**可能原因：**
- 115网盘未配置
- 网络连接问题
- 115 API限制

**解决方法：**
1. 检查115网盘配置（设置 -> 媒体设置）
2. 查看错误信息
3. 稍后重试

### 问题3：捕获了不需要的资源
**解决方法：**
1. 添加排除关键词
2. 使用批量忽略功能
3. 调整规则配置

## 🎯 最佳实践

1. **精确的关键词**
   - 使用具体的关键词，避免过于宽泛
   - 善用排除关键词过滤低质量资源

2. **合理的目标路径**
   - 按类型分类：`/电影`、`/剧集`、`/纪录片`
   - 按来源分类：`/来源/电影群`、`/来源/资源频道`
   - 按时间分类：`/电影/2024`

3. **自动转存的使用**
   - 对于信任的来源，启用自动转存
   - 对于不确定的资源，先手动审核

4. **定期清理**
   - 删除已处理的记录
   - 忽略不需要的资源
   - 优化规则配置
